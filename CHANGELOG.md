# Список изменений

## 17–22 марта 2026

### Финансы и абонементы
- **Абонементы** — добавлена система абонементов: название, количество уроков, цена. Каждому ученику можно назначить абонемент
- **Баланс студентов** — автоматическое списание за каждый проведённый урок из баланса ученика
- **Сроки абонемента** — поля «действует с» и «действует до»
- **Скидки** — скидка для студента: фиксированная (в рублях) или процентная, с периодом действия
- **Зарплаты учителей** — автоматическое начисление зарплаты при проведении урока (по ставке учителя)
- **Вкладка «Ожидают оплаты»** — список студентов с долгом или ≤ 2 оставшихся уроков, долги отображаются первыми

### Студенческий портал (новое приложение)
- Запущен отдельный клиент для учеников по адресу **web.rancheasy.ru**
- **Авторизация** — вход по логину/паролю (отдельно от CRM)
- **Главная страница** — расписание, посещаемость, баланс уроков
- **Экзамены** — список доступных экзаменов с возможностью онлайн-записи
- **Результаты** — просмотр оценок и результатов экзаменов
- Печать логинов/паролей для учеников из CRM

### Экзамены (CRM)
- Управление записью на экзамены: открыть/закрыть регистрацию
- Просмотр записавшихся учеников
- Связь экзаменов с предметом группы
